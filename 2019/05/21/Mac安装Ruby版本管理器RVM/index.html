Mac安装Ruby版本管理器RVM

发表于 2019-05-21 分类于 资料

简介

RVM是一个命令行工具，它允许您轻松地安装，管理和使用从解释器到多组宝石的多个ruby环境。

安装Ruby原因

虽然 macOS 自带了一个 ruby 环境，但是是系统自己使用的，所以权限很小，只有 system。而/Library 目录是 root 权限,所以很多会提示无权限。
